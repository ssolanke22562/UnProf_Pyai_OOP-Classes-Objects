# 🎓 Student Report Card Management System

## 📌 Overview

The Student Report Card Management System is a menu-driven Python application developed using Object-Oriented Programming (OOP) concepts. The system allows users to create a student profile, add grades for different subjects, update existing grades, and display a formatted report card.

This project demonstrates the practical implementation of Python OOP concepts such as Classes, Objects, Inheritance, Constructors, and Dictionaries.

---

## 🚀 Features

* Create a student profile
* Add grades for multiple subjects
* Update existing grades
* Display a formatted report card
* Menu-driven user interface
* Uses inheritance to promote code reusability

---

## 🛠️ Technologies Used

* Python 3

---

## 📚 OOP Concepts Implemented

### 1. Classes and Objects

* `Person` class
* `Student` class

### 2. Inheritance

* `Student` inherits properties from `Person`

### 3. Constructors

* Uses `__init__()` method for object initialization

### 4. Methods

* Add Grade
* Update Grade
* Display Report Card

### 5. Dictionaries

* Stores subjects and grades efficiently
